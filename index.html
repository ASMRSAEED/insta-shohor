Question & Answer:

Question-1: How does Javascript work?

Answer:- JavaScript is an interpreted and scripting language. It can implement complex features and update them dynamically. Also, it can modify HTML, CSS dynamically. It can validate complex entries. Javascript can't run without browser engines like V8, JavaScriptCore or SpiderMonkey. JavaScript runs from top to bottom sequentially.

Question-2: What does javascript event loop do?

Answer:- An event loop is a process that monitors the callback queue and the call stack. Call back queue holds the result which has setTimeout function and execute later. In the meantime, the call stack executes the other function. The event loop waits until processing all of the functions are from the call stack. After executing all of the functions from the call stack and when the call stack is empty the event loop forward the event into the call stack which has setTimeout function and supposes to execute later for processing.

Question-3: What's the difference between local storage and session storage?

Answer:- Local storage and session storage both are store data and they store data as a key-value pair. Local storage stores data long time until it's removed by the user. Other hand, season storage stores data until the season ends. local storage has much greater store capacity against season storage. Local storage store capacity is 10MB and season storage store capacity is 5MB. Local storage stores data into the browser and server but season storage stores data only into the browser.
